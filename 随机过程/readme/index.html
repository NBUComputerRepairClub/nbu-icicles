期末考试

期末考试的组成如下

  • 叙述题 25分
  • 填空题 15分
  • 计算题 40分（5个）
  • 证明题 20分（2个）

范围是1-9章，第四章更新过程不考。

以下是个人回忆的期末考题：

简答题x5（写概念）

  • 条件期望的性质
  • possion过程
  • 首达概率
  • 正常返
  • 离散鞅

填空题（忘了不全）

  • $N(s+h)-N(s)$ 概率服从一个$2\times2$的表格，求条件期望
  • 0-1布朗运动，求期望$E(X(S)X(T))$

大题（忘了不全）

  • 求$P{\int^1_0B(t)\mathrm{d}t}<\frac{2}{\sqrt{3}}$的概率
  • 伊藤公式计算
  • possion过程里的指数推导
  • 证明布朗运动的鞅是自己
