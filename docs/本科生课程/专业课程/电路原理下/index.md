## 课程学习内容
+ 这门课是电信类除通信工程外的其他三个专业都要修的一门课，俗称“电路下”，是在电路上（特别是相量法）的基础上 上的课
+ 挂科率比较高，公认的难，而且要用到复数，所以建议认真听课（不然挂科的时候也是认真的）
+ 考试的时候可以带计算器的，所以建议买个好一点的计算器（最好是能进行复数运算的复数计算器）
+ 具体的复习资料没有多少，重点是看老师的 PPT 和作业
